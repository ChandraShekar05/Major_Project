About This Project

This project is a Credit Card Fraud Detection system built using Autoencoders in Keras. The primary goal of this project is to identify fraudulent transactions in credit card datasets by leveraging machine learning techniques.

Key Features

  • Uses Autoencoders, a type of neural network, for anomaly detection.
  • Supports two datasets: creditcard.csv and filtered_creditcard.csv.
  • Provides statistical analysis of the datasets, including visualizations such as pie charts and bar graphs.
  • Allows users to fetch transaction details and predict whether a transaction is fraudulent.

Technologies Used

  • Frontend: HTML, CSS, JavaScript, Chart.js
  • Backend: Flask (Python)
  • Machine Learning: Keras (TensorFlow)
  • Data Visualization: Chart.js

How It Works

The system uses an Autoencoder model trained on normal transactions to detect anomalies. Transactions that deviate significantly from the normal pattern are flagged as fraudulent. Users can interact with the system to fetch transaction details, view dataset statistics, and predict the likelihood of fraud for specific transactions.

Purpose

This project demonstrates the application of machine learning in fraud detection, providing insights into how data-driven solutions can enhance security in financial systems.
